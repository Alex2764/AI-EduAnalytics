# Училищна система за управление на тестове

Уеб приложение за управление на класове, ученици и техните резултати от тестове. Направено е за да улесни работата на учителите при водене на оценки и анализ на успеваемостта.

## Какво прави

- Създаване и управление на класове
- Добавяне на ученици (поотделно или масово)
- Създаване на различни видове тестове (класни работи, входно/изходно ниво и т.н.)
- Въвеждане на резултати и автоматично изчисляване на оценки
- Статистика и анализ на успеваемостта (по ученици, класове и тестове)
- Графики за проследяване на напредъка

## Технологии

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (за база данни)

## Как да стартирам

```bash
npm install
npm run dev
```

Приложението ще се отвори на `http://localhost:5173`

## Build

```bash
npm run build
```
